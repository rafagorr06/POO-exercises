Diseñe una clase genérica llamada Calculos que tenga como atributos un entero n (tamaño del vector), un vector de n elementos genéricos y 2 datos de mismo tipo genérico que los elementos del vector. La clase debe tener un constructor, un método llamado dosmax(..) que obtenga los 2 datos mayores del vector y métodos que considere necesarios. En un programa cliente instancie la clase para una lista de flotantes y para una lista de strings que deben leerse como datos.
